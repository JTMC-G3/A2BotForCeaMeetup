# A2BotForCeaMeetup
A remake of the A2 Bot for the CEA Meetup



# Features:
1. howmanyspace/hotmanspacemike
   Simply shows how many users are in the closed ea meetup server at a time
2. ?naping
   Allows users to ping for people to get on, similar to ea-scheduling but now back.
